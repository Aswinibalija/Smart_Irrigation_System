🧠This project is a Smart Irrigation System that uses real-time sensor data (moisture, temperature, and humidity) to automate water delivery in fields, promoting efficient water use and sustainable agriculture.

🛠️It is developed using Python (Django) for the backend, along with HTML, CSS, JavaScript, Bootstrap for the frontend. The backend is powered by a REST API that feeds data dynamically into the dashboard.

📈The system supports:

  • 🔍 Real-time monitoring of soil and environment
  • 🗃️ Historical data analysis
  • 📲 Remote irrigation control (via MQTT)
  • 📤 RESTful data communication between sensors and server

🚀Overall, this is a great example of integrating IoT with modern web development to solve real-world agricultural challenges.
